Get started in 60 seconds

Install, analyze, explore. No signup, no telemetry, runs locally.

# 1. Install
npm install -g archlens-studio

# 2. Analyze any project
cd your-project
archlens-studio analyze .

# 3. Open the dashboard
archlens-studio serve
# → http://localhost:4848
